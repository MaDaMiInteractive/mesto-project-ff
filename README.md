# Mesto

Учебный проект с редактированием профиля, карточками мест, валидацией форм и интеграцией с API.

## Стек

- HTML
- CSS
- JavaScript
- Webpack
- Babel
- PostCSS

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Публикация

Для деплоя на GitHub Pages используется команда:

```bash
npm run deploy
```
